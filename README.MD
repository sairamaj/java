# Collections
* package
    * java.util.*
* interface
    * Collection<E>
* super class
    * AbstractCollection<E>

| Name                    |  Purpose         |  Backend    |    Access
|------------------------ |----------------- |-------------|-----------
| AbstractList            |  Unmodified list |  Array      |   Random
| AbstractSequentialList  |  Unmodified list |  Linked List|   Sequential
